# Bern (.brn) Syntax + IntelliSense Extension

## Overview

This Visual Studio Code extension provides syntax highlighting and IntelliSense support for the Bern programming language (`.brn` files). It is designed to enhance the development experience for Bern language users by offering:

- Syntax highlighting for `.brn` files
- IntelliSense and basic code completion
- Language configuration for improved editing
- Easy identification with a cat icon

## Features
- Automatic syntax highlighting for Bern source files
- Language support for `.brn` file extensions
- Lightweight and easy to use

## Installation
1. Download or clone this repository.
2. Open the folder in Visual Studio Code.
3. Press `F5` to launch an Extension Development Host, or install from the VSIX package.
